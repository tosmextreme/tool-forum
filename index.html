📋 วิธีสร้าง Google OAuth Client ID
1
เปิด console.cloud.google.com → New Project
2
APIs & Services → OAuth consent screen → External → Save
3
Credentials → + Create Credentials → OAuth client ID → Web application
4
Authorized JavaScript origins → เพิ่ม URL ของไซต์ เช่น https://yourdomain.com
5
Copy Client ID (ลงท้ายด้วย .apps.googleusercontent.com) → วางด้านล่าง
⚠️ Client ID ไม่ถูกต้อง
💡 Client ID จะถูกเก็บใน Browser (localStorage) บน Browser นี้เท่านั้น
